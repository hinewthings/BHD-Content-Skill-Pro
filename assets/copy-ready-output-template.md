# Copy-Ready Output Template

Use this template when the user wants content they can copy and use immediately.

Rules:

```txt
1. Keep strategy notes outside the copy-ready block.
2. Inside the copy-ready block, include only final user-facing content.
3. Do not include labels such as Headline, CTA, Hook, Primary text, Caption, Description, or Notes.
4. If platform requires separate fields, place field labels outside the copy block, then provide the clean copy below each field.
5. Avoid explanations inside the copy-ready block.
```

Suggested output structure:

```txt
Ghi chú ngắn:
[strategy/compliance notes if needed]

BẢN CÓ THỂ COPY DÙNG NGAY
---
[final user-facing content only]
---

BIẾN THỂ KHÁC
---
[final user-facing content only]
---
```
