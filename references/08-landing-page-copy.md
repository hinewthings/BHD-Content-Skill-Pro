# Landing Page Copy

Use this reference for landing page structure and copy.

## Recommended Blocks

```txt
1. Hero: pain, promise, audience, CTA
2. Problem: clarify current pain
3. Agitation: show cost of inaction
4. Solution: introduce the offer
5. Mechanism: explain why it works
6. Benefits: translate features into outcomes
7. Proof: case study, demo, process, numbers, testimonials
8. Offer: what is included
9. FAQ / Objection: handle doubts
10. Final CTA: clear next action
```

## Landing Page Strategy

```txt
If audience is problem aware:
Lead with problem clarity and diagnosis.

If solution aware:
Lead with mechanism and why this approach is better.

If product aware:
Lead with proof, offer clarity, and risk reversal.

If most aware:
Lead with direct offer, deadline, bonus, and CTA.
```

## Output Contract

Return:

```txt
Strategic header
Hero headline
Hero subheadline
Primary CTA
Problem section
Agitation section
Solution section
Mechanism section
Benefit bullets
Proof section
Offer stack
FAQ / objection section
Final CTA
Suggested visual/proof assets
Compliance notes
```

## Copy Rules

```txt
Do not overpromise.
Do not hide the mechanism.
Use proof near major claims.
Make CTA specific.
Use section headlines that say something, not generic labels.
```
