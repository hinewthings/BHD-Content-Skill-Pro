# Repurpose & Calendar

Use this reference to turn one idea or source into many assets and to build a channel system.

## Repurpose Factory

One strong source can become:

```txt
5 short posts
5 short video scripts
3 carousel outlines
3 ads angles
1 email
1 landing page section
1 checklist
1 FAQ
1 case study outline
```

## Repurpose Process

```txt
1. Extract main insight
2. Extract sharp quotes
3. Extract proof or proof gaps
4. Extract story
5. Extract objections
6. Extract CTA
7. Convert by format and journey stage
```

## Channel System

Define:

```txt
Channel positioning
Primary audience
Main topics
3-5 content pillars
Repeatable series
Formats
Brand voice
Posting rhythm
Engagement mechanism
Conversion mechanism
```

## Weekly Calendar Pattern

```txt
Monday: POV / insight
Tuesday: How-to / checklist
Wednesday: Case study / proof
Thursday: Debate / myth bust
Friday: Story / behind the scenes
Saturday: Offer / soft sell
Sunday: Recap / comment response / community
```

## Calendar Output Contract

Return:

```txt
Channel goal:
Audience:
Pillars:
Posting rhythm:
Calendar table:
Each item: date/day, pillar, format, hook, angle, journey stage, CTA, proof needed
Repurpose plan:
```
